# Hi, I'm Prabhav 

Right now Im building a **Parkinsons UPDRS forecasting app** (XGBoost + Streamlit) that turns routine clinic scores into short-term symptom forecasts. The goal of this project is to help families see the **near-term symptom burden** coming upuseful for planning day-to-day life and for **earlier conversations with clinicians**. I focus on **UPDRS I** (non-motor symptoms) because changes there can flag broader needssleep hygiene, mood support, or a medication revieweven when motor scores look stable. *Educational tool, not medical advice.*
